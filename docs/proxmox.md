<h1 align="center"> Proxmox Scipts </h1>

### Import OVA

- Import Virtual Appliance into Proxmox
```shell
# Pull Script

curl -O https://raw.githubusercontent.com/jershbytes/scripts/main/proxmox/import-ova.sh

# Run Script
./import-ova.sh
```
